# 修复自定义规则中的错误

> 此篇依旧为实战篇，当我们的自定义规则生效后，我们如何自动修复它呢？

## 目标

- 使用自定义规则进行lint,不允许使用过时的函数调用
- 使用fix 属性修复我们的自定义规则lint错误

## Coding

- 对我们的自定义规则做一下小小的修改，将AST中的函数调用表达式拿到，然后进行处理
- 在导出的函数中添加meta fixable属性，表示这个自定义规则可被修复
- 使用eslint 提供的fixer api去进行修复
